# GoogleConsole_CPUminer
### install 
- ```git clone https://github.com/samsesh/GoogleConsole_CPUminer.git```
- ```cd GoogleConsole_CPUminer```
- ```chmod +x install.sh```
- ```sudo install.sh```
### run
- edit run.sh for config 
- ```nano run.sh```
- ```chomd +x run.sh ```
- then run ```sudo run.sh``` for mining
